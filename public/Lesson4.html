Position Property

  • static, relaitve, absolute and fixed
  • static is the default of all elements
  • adjustable position to top right bottom left
  • can remove the element from the flow of the document
  • can be relative to certain things
  • absolute and fixed will allow the top, left, right bottom
  • Button is relative to the element
  • Tooltip is positioned absolute so it is out of the flow and display = hidden
  • Tooltip display is block when on hover
    html#relativeAbsolute .btn:hover .tooltips {
      display: block;
      bottom: 38px;
      left: 10px;
    }
