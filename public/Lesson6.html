Flexbox

  • specify display property on a parent to control child elements will display to that parent
  • can specify a specific child but mostly work with the parent element

Horizontal Alignment

  • Parent-display: flex;
  • then the children will be a flex component
  • flex-direction: DIR;
    • default: row, column, row-reverse, column-reverse
  • flex-wrap: WRP;
    • specify if the items wrap (break to another line or just compress)
    • default: nowrap, wrap, wrap-reverse
  • justify-content: JUS;
    • default: flex-start which is left aligned, etc
